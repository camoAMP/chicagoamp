# Chicago AMP

Chicago AMP is a modern Next.js 16 application built with TypeScript, featuring a comprehensive business website for creative services including video production, marketing, web development, and other digital services.

## Features

- Next.js 16 with App Router
- React 19 with TypeScript
- Tailwind CSS for styling
- Radix UI primitives for accessible components
- Framer Motion for smooth animations
- Responsive design for all devices
- Dark/light mode support
- Deployment ready for Cloudflare Pages

## Services Offered

The platform showcases various services including:
- Video Production & Commercial Video
- Marketing & Digital Marketing
- Website Development
- SEO Services
- Sound Design
- Lighting & Set Design
- Event Coverage
- Wedding Films
- Graphics & Branding
- Music Videos

## Tech Stack

- **Framework**: Next.js 16 (with Turbopack)
- **Language**: TypeScript
- **Styling**: Tailwind CSS with `tailwindcss-animate`
- **UI Components**: Radix UI Primitives
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Charts**: Recharts
- **Maps**: Leaflet with React-Leaflet
- **Forms**: React Hook Form with Zod validation
- **Authentication**: NextAuth.js
- **Deployment**: Cloudflare Pages

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
